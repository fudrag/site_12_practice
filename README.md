Функциональность
Основные разделы сайта: 
1. Header (Шапка) 
- Логотип и название компании 
- Навигационное меню 
- Закрепленная позиция при скролле 
2. Hero Section (Главный баннер) 
- Приветственный текст 
- Кнопка призыва к действию 
3. About Section (О нас) 
- Информация о компании 
- Изображение продукта 
4. Catalog Section (Каталог) 
- Сетка товаров (3 продукта)
- Цены и описания 
- Кнопки "Подробнее" 
5. Customization Section (Возможности кастомизации) 
- Опции персонализации столов 
- Три категории: цвет, декор, дополнительные опции 
6. Gallery Section (Галерея и «работы» фирмы) 
- Фотографии готовых рабочих пространств 
- Эффекты наведения 
7. Footer (Нижняя часть с информацией) 
- Контактная информация 
- Социальные сети 
- Копирайт

Установка и запуск
1. Скачайте все файлы проекта в одну папку 
2. Убедитесь, что структура папок сохранена 
3. Откройте файл index.html в браузере 
Требования к изображениям
Для корректной работы сайта необходимы следующие изображения в папке images/:
- лого.jpg (логотип компании) 
- фембой.png (изображение стола в стиле милого мальчика) 
- 1 стол.jpg, 2 стол.jpg, 3 стол.jpg (изображения товаров) 
- цвет.png, декор.png, доп.опции.png (иконки для кастомизации) 
- рабочая зона.png, яркое рабочее место.png, стильный стол для творчества.png (галерея)
